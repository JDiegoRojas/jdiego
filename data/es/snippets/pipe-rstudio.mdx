## Introducción
¿Te gustaría simplificar tus flujos de trabajo en RStudio? ¡Entonces estás en el lugar correcto! El operador |> es una herramienta poderosa de la librería base de R que te permite encadenar operaciones de manera fluida y elegante.

### ¿Qué es el operador pipe '|>'?
El operador |> (también conocido como "pipe forward") te permite pasar el resultado de una expresión como el primer argumento de la siguiente expresión. Esto hace que tu código sea más fácil de leer y entender.

## Instalamos los paquetes y cargamos las librerías
Para usar el operador pipe en RStudio, necesitas tener instalada la versión ``4.1.0`` o superior de R. Además, necesitarás cargar la librería base de R, que incluye el operador pipe. Para manipular datos, también puedes cargar la librería ``dplyr``.

```r
# Instalar y cargar las librerías necesarias

install.packages("dplyr")

library(dplyr)

```


##  Ejemplo sin el operador pipe

#### Cargar el conjunto de datos

Primero, carguemos el conjunto de datos ``mtcars``. Este conjunto de datos contiene mediciones de 11 atributos diferentes para 32 automóviles. Utilizaremos funciones como ``filter``, ``select``, ``summary`` y crearemos gráficos para explorar los datos.

```r
# Cargar el conjunto de datos mtcars
data(mtcars)

# Ver las primeras filas del conjunto de datos
head(mtcars)

```

#### Filtrar y seleccionar
Supongamos que queremos analizar solo los autos con un consumo de combustible (mpg) superior a 20. Filtraremos y seleccionaremos las columnas relevantes:
```r


# Filtrar autos con más de 150 caballos de fuerza
filtered_cars <- filter(mtcars, hp > 150)

# Seleccionar solo las columnas mpg, hp y wt
selected_columns <- select(filtered_cars, mpg, hp, wt)

# Calcular la media de millas por galón (mpg), caballos de fuerza (hp) y peso (wt)
avg_mpg <- mean(selected_columns$mpg)
avg_hp <- mean(selected_columns$hp)
avg_wt <- mean(selected_columns$wt)

# Imprimir los resultados
print(avg_mpg)
print(avg_hp)
print(avg_wt)

```


## Repitiendo lo mismo con el operador |>
Ahora, hagamos lo mismo utilizando el operador |>:

```r
library(dplyr)

# Cargar el dataset mtcars y realizar operaciones en cadena usando el operador pipe
result <- mtcars |>
  # Filtrar autos con más de 150 caballos de fuerza
  filter(hp > 150) |>
  # Seleccionar solo las columnas mpg, hp y wt
  select(mpg, hp, wt) |>
  # Calcular la media de millas por galón (mpg), caballos de fuerza (hp) y peso (wt)
  summarise(avg_mpg = mean(mpg), avg_hp = mean(hp), avg_wt = mean(wt))

# Imprimir el resultado
print(result)

```

Ambos ejemplos logran el mismo resultado, pero el segundo ejemplo utiliza el operador pipe '|>' para encadenar las operaciones, lo que hace que el código sea más conciso y legible.
